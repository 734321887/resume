## 科研与论文

**RepoContraAudit: Explainable Repository-Level Code Auditing via Multimodal Anti-Fact Reasoning**

- **状态：** NeurIPS 投稿中　｜　**角色：** 通讯作者
- **研究方向：** 仓库级代码审计、多模态推理、可解释性反事实验证
- **问题定义：** 面向真实代码仓库中的复杂缺陷与安全隐患，探索可解释、可核验的自动化审计方法
- **主要工作：** 主导研究路线与整体框架设计，提出 Multimodal Anti-Fact 推理机制，构建从审计假设生成、证据检索到反事实验证的完整链路；与研究员协作完成数据集构建、实验设计与论文撰写
- **技术要点：** 代码库级上下文理解、跨文件依赖分析、工具调用链验证、反事实检验与可解释性输出
